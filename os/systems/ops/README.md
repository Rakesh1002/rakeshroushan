# Ops — Meetings, Decisions, Notes

Operational documents that don't belong to a specific project.

## Structure

- `meetings/` — meeting notes by date (`YYYY-MM-DD-topic.md`)
- Decisions, strategy discussions, vendor evaluations, etc.

## Meeting note template

```markdown
# YYYY-MM-DD — Meeting Topic

**Attendees:**
**Duration:**

## Agenda

1.
2.

## Notes

## Decisions

- [ ] Decision 1 → Owner, Due date
- [ ] Decision 2 → Owner, Due date

## Next Actions

- [ ] Action 1 → Owner, Due date
```
